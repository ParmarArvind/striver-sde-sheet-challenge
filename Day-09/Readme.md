# 📅 Day 09 - Striver SDE Sheet Challenge

## ✅ Problems Solved

### 1. Reverse Linked List (LeetCode 206)

- **Approach:** Iterative (Three Pointers) & Recursive
- **Time Complexity:** O(N)
- **Space Complexity:**
  - Iterative: O(1)
  - Recursive: O(N)

**Key Learning:**
- Practiced reversing a linked list using both iterative and recursive approaches.
- Understood how the `prev`, `curr`, and `next` pointers work together.
- Learned how recursion naturally reverses the links while returning from recursive calls.

---

### 2. Middle of the Linked List (LeetCode 876)

- **Approach:** Slow & Fast Pointer
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)

**Key Learning:**
- Strengthened the Slow & Fast Pointer technique.
- Learned how moving one pointer twice as fast automatically finds the middle node.
- Also understood why the second middle node is returned for even-length lists.

---

### 3. Merge Two Sorted Lists (LeetCode 21)

- **Approach:** Dummy Node + Iteration
- **Time Complexity:** O(N + M)
- **Space Complexity:** O(1)

**Key Learning:**
- Learned how a dummy node simplifies linked list problems.
- Avoided special handling for the head node.
- Improved confidence in pointer manipulation while merging two sorted lists.
