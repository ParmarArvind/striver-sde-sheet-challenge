### 2. Max of Min for Every Window Size (GFG)

- **Approach:** Monotonic Stack (Nearest Smaller Left & Right)
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)

**Key Learning:**

- Computed nearest smaller elements on both sides.
- Determined maximum window for each minimum.
- Filled unanswered window sizes using backward propagation.

### 3. The Celebrity Problem (GFG)

- **Approach:** In-degree & Out-degree Counting
- **Time Complexity:** O(N²)
- **Space Complexity:** O(N)

**Key Learning:**

- Used indegree/outdegree observations.
- Verified celebrity conditions directly.
- Reinforced matrix traversal concepts.

### 4. Online Stock Span (LeetCode)

- **Time Complexity:** O(N) Amortized
- **Space Complexity:** O(N)
