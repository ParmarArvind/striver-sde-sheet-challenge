### 3. Implement Stack using Queues (LeetCode)

- **Time Complexity:**

**Key Learning:**

### 4. Implement Queue using Stacks (LeetCode)

- **Approach:** Two Stacks
- **Time Complexity:**
  - push() → O(1)
  - pop() → Amortized O(1)
  - peek() → Amortized O(1)
- **Space Complexity:** O(N)

**Key Learning:**
- Used two stacks to implement queue operations.
- Deferred element transfers until necessary for better efficiency.
- Understood amortized time complexity in practical implementations.
