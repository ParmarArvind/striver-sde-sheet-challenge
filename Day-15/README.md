# 📅 Day 15 - Striver SDE Sheet Challenge

## ✅ Problems Solved

### 1. Minimum Platforms (GFG)

- **Approach:** Sorting + Two Pointers
- **Time Complexity:** O(N log N)
- **Space Complexity:** O(1)

**Key Learning:**
- Learned how sorting arrival and departure times simplifies interval scheduling.
- Used two pointers to efficiently count the maximum platforms required.
- Strengthened understanding of greedy interval problems.

---

### 2. Job Sequencing Problem (GFG)

- **Approach:** Sorting + Min Heap (Priority Queue)
- **Time Complexity:** O(N log N)
- **Space Complexity:** O(N)

**Key Learning:**
- Learned how a priority queue helps maximize total profit.
- Understood the importance of sorting jobs by deadlines.
- Practiced combining greedy algorithms with heaps.

---

### 3. Fractional Knapsack (GFG)

- **Approach:** Greedy (Value/Weight Ratio)
- **Time Complexity:** O(N log N)
- **Space Complexity:** O(N)

**Key Learning:**
- Learned why selecting items based on value per unit weight is optimal.
- Practiced implementing fractional selection.
- Strengthened greedy optimization concepts.

---

### 4. Activity Selection (GFG)

- **Approach:** Greedy + Sorting
- **Time Complexity:** O(N log N)
- **Space Complexity:** O(N)

**Key Learning:**
- Learned the classic activity selection greedy strategy.
- Understood why sorting by finishing time gives the optimal answer.
- Improved confidence in solving scheduling problems.
