# 📅 Day 11 - Striver SDE Sheet Challenge

## ✅ Problems Solved

### 1. Intersection of Two Linked Lists (LeetCode 160)

- **Approach:** Hash Set & Two Pointer Switching
- **Time Complexity:** O(N + M)
- **Space Complexity:**
  - Hash Set: O(N)
  - Two Pointers: O(1)

**Key Learning:**
- Started with the Hash Set approach to quickly identify the intersection node.
- Learned the elegant Two Pointer Switching technique that achieves the same time complexity with O(1) extra space.
- Understood how traversing both lists equalizes the path length for both pointers.

---

### 2. Linked List Cycle (LeetCode 141)

- **Approach:** Hash Set & Floyd's Cycle Detection (Slow & Fast Pointer)
- **Time Complexity:** O(N)
- **Space Complexity:**
  - Hash Set: O(N)
  - Floyd's Algorithm: O(1)

**Key Learning:**
- Revised the Hash Set approach for detecting revisited nodes.
- Learned Floyd's Tortoise and Hare algorithm for cycle detection without extra memory.
- Understood why the slow and fast pointers must eventually meet if a cycle exists.

---

### 3. Reverse Nodes in K-Group (LeetCode 25)

- **Approach:** Recursive K-Group Reversal
- **Time Complexity:** O(N)
- **Space Complexity:** O(N/K) (Recursive Call Stack)

**Key Learning:**
- Practiced reversing linked list nodes in fixed groups of size **k**.
- Learned how recursion can simplify connecting the reversed groups.
- Improved confidence in handling complex pointer manipulation problems.
