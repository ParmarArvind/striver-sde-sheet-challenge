## ✅ Problems Solved

### 1. Median of Two Sorted Arrays (LeetCode)

- **Approach:** Binary Search on Partitions
- **Time Complexity:** O(log(min(N, M)))
- **Space Complexity:** O(1)

**Key Learning:**
- Learned how to partition two sorted arrays using Binary Search.
- Understood the conditions for finding the correct partition.
- Solved the median problem without merging the arrays, achieving optimal complexity.

### 2. Aggressive Cows (GFG)

- **Approach:** Binary Search on Answer + Greedy
- **Time Complexity:** O(N log(MaxDistance))
- **Space Complexity:** O(1)

**Key Learning:**
- Applied Binary Search on the answer space instead of array indices.
- Used a greedy feasibility check to validate each candidate distance.
- Strengthened understanding of optimization problems using Binary Search.

---

### 3. K-th Element of Two Sorted Arrays (GFG)

- **Approach:** Binary Search on Partitions
- **Time Complexity:** O(log(min(N, M)))
- **Space Complexity:** O(1)

**Key Learning:**
- Extended the partition-based Binary Search technique from the median problem.
- Learned how to identify the k-th element without merging both arrays.

### 4. Allocate Minimum Pages (GFG)

- **Approach:** Binary Search on Answer + Greedy
- **Time Complexity:** O(N log(Sum of Pages))
- **Space Complexity:** O(1)

**Key Learning:**
- Used Binary Search to minimize the maximum pages assigned to a student.
- Implemented a greedy validation function for checking feasible allocations.
- Practiced solving allocation and scheduling problems using Binary Search.
