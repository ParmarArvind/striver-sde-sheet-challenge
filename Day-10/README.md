# Day 10 – Linked List I

## ✅ Problems Solved

### 1. Remove Nth Node From End of List

* **Platform:** LeetCode
* **Approach:** Two Pointer
* **Time Complexity:** O(N)
* **Space Complexity:** O(1)

**Key Learning:**

* Maintained a fixed gap between two pointers to locate the target node in a single traversal.

---

### 2. Add Two Numbers

* **Platform:** LeetCode
* **Approach:** Simulation with Dummy Node
* **Time Complexity:** O(max(N, M))
* **Space Complexity:** O(max(N, M))

**Key Learning:**

* Simulated digit-by-digit addition while handling carry, similar to manual addition.

---

### 3. Delete Node in a Linked List

* **Platform:** LeetCode
* **Approach:** Overwrite Next Node
* **Time Complexity:** O(1)
* **Space Complexity:** O(1)

**Key Learning:**

* Learned the special trick of deleting a node without access to its previous node by copying the next node's value.

---

## 📌 Day 10 Summary

* Practiced three important linked list interview problems.
* Revised Two Pointer and Dummy Node techniques.
* Learned a unique constant-time deletion trick based on problem constraints.
