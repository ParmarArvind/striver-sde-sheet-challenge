# 📅 Day 13 - Striver SDE Sheet Challenge

## ✅ Problems Solved

### 1. Copy List with Random Pointer (LeetCode 138)

- **Approach:** Hash Map
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)

**Key Learning:**
- Learned how to clone complex linked lists containing random pointers.
- Used a hash map to maintain the mapping between original and copied nodes.
- Understood how multiple traversals simplify pointer assignments.

---

### 2. 3Sum (LeetCode 15)

- **Approach:** Sorting + Two Pointers
- **Time Complexity:** O(N²)
- **Space Complexity:** O(1) (excluding output)

**Key Learning:**
- Revised the Two Pointer technique on sorted arrays.
- Learned how to efficiently avoid duplicate triplets.
- Strengthened understanding of fixing one element and searching for the remaining pair.

---

### 3. Rotate List (LeetCode 61)

- **Approach:** Array Rotation & Circular Linked List
- **Time Complexity:** O(N)
- **Space Complexity:**
  - Array: O(N)
  - Circular List: O(1)

**Key Learning:**
- Initially solved the problem using an auxiliary array.
- Optimized it by making the linked list circular.
- Learned how to identify the new head and tail after rotation.
