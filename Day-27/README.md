# 📅 Day 27 - Striver SDE Sheet Challenge

## ✅ Problems Solved

### 1. Largest Rectangle in Histogram (LeetCode)

- **Approach:** Monotonic Stack (Nearest Smaller Left & Right)
- **Time Complexity:** O(N)
- **Space Complexity:** O(N)

**Key Learning:**
- Used monotonic stacks to compute the nearest smaller element on both sides.
- Learned how rectangle width is determined using boundary indices.
- Strengthened understanding of stack-based optimization techniques.

---

### 2. Sliding Window Maximum (LeetCode)

- **Approach:** Max Heap + Hash Map
- **Time Complexity:** O(N log K)
- **Space Complexity:** O(K)

**Key Learning:**
- Applied a priority queue to maintain the maximum element in each window.
- Used lazy deletion with a hash map to efficiently discard expired elements.
- Practiced solving sliding window problems using heap-based techniques.

---

### 3. Min Stack (LeetCode)

- **Approach:** Two Stacks
- **Time Complexity:** O(1) per operation
- **Space Complexity:** O(N)

**Key Learning:**
- Implemented constant-time minimum retrieval using an auxiliary stack.
- Learned how to synchronize push and pop operations across two stacks.
- Reinforced stack implementation concepts.
